# azix

CLI for Azix source-copy workflow.

Use it to initialize a project and copy editable component source into local `components` folders.

## Quick start

```bash
npx azix init --template next --defaults
npx azix add button input
```

## Main commands

```bash
npx azix list
npx azix add button input data-table
npx azix preset minimal
npx azix theme src/index.css
```
